Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are represented by tags.
  • HTML elements can have attributes.
  • HTML elements can be nested.
  • HTML elements can be empty or contain content.
  • The metadata describes information about the application

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets.
  • CSS is used to style HTML elements.

Git

  • Git is a version control system.
  • Git is used to track changes in the code.
  • Git is used to collaborate with other developers.

JavaScript

  • JavaScript is a programming language.
  • JavaScript is used to add interactivity to a webpage.
  • JavaScript is used to create dynamic webpages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
